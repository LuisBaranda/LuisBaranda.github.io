Siglas

OSI (Open Systems Interconnection)

Modelo de referencia para entender y diseñar una red de comunicación en siete capas

IP (Internet Protocol)

Protocolo que dirige el tráfico de internet mediante la asignación de direcciones IP

TCP (Transmission Control Protocol)

Protocolo que garantiza la entrega de datos en orden y sin errores en una red IP

UDP (User Datagram Protocol)

Protocolo de comunicación más rápido que TCP, pero que no garantiza la entrega de datos

SSH (Secure Shell)

Protocolo de red para acceder de forma segura a dispositivos remotos

FTP (File Transfer Protocol)

Protocolo para transferir archivos entre un cliente y un servidor en una red

SMTP (Simple Mail Transfer Protocol)

Protocolo usado para enviar correos electrónicos entre servidores

POP (Post Office Protocol)

Protocolo para recibir correos electrónicos, generalmente usado para descargar correos a un cliente local

DHCP (Dynamic Host Configuration Protocol)

Protocolo que asigna automáticamente direcciones IP a dispositivos en una red

DNS (Domain Name System)

Sistema que traduce nombres de dominio en direcciones IP

HTTP (Hypertext Transfer Protocol)

Protocolo usado para transferir datos en la web

ARP (Address Resolution Protocol)

Protocolo que traduce direcciones IP a direcciones MAC en una red local

MAC (Media Access Control)

Dirección física única asignada a cada dispositivo de red

LAN (Local Area Network)

Red de computadoras en un área geográfica limitada, como una oficina o un hogar
